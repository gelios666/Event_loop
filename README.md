# Загрузка персонажей Star Wars в базу данных

Асинхронный скрипт для выгрузки всех персонажей из SWAPI (Star Wars API) и сохранения их в PostgreSQL.

## Требования
- Python 3.9+
- PostgreSQL (можно запустить через Docker)

## Установка и запуск

1. Клонировать репозиторий
2. Установить зависимости 
3. Запустить PostgreSQL
4. Выполнить миграцию (создание таблицы)
5. Загрузить данные
